SVG (Scalable Vector Graphics): SVG is a vector-based graphics format that is defined in XML (Extensible Markup Language). SVG graphics are resolution-independent and can be scaled without a loss of quality. You can embed SVG graphics directly into HTML using the "svg" element or use them as standalone SVG files.
