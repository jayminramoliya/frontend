• What is React Js?
	-React, is an open-source JavaScript library developed and maintained by Facebook.
	-React is a User Interface (UI) library
	-React is a tool for building UI components

• What is NPM in React Js?
	-Node Package Manager
	-NPM, the Node package manager is used for managing and sharing the packages for either React or Angular.

• What is Role of Node Js in react Js?
	-Node.js plays several important roles in the React.js development ecosystem. Here are some key aspects of the role of Node.js in React.js development:
	Package Management,Development Server,Build Tools,Middleware and API Servers,Proxy Servers for API Requests,Universal (Isomorphic) JavaScript,Deployment

• What is CLI command In React Js?
	-React has its own command-line interface (CLI) commands. However, these CLI commands are currently only used to create a passable version of a react application using the command 	line.

• What is Components in React Js?
	-Components are independent and reusable bits of code. They serve the same purpose as JavaScript functions, but work in isolation and return HTML. Components come in two types, Class 	components and Function components, in this tutorial we will concentrate on Function components.

• How to install React Js on Windows, Linux Operating System? How to Install NPM and How to check version of NPM?
	-Installing React.js and NPM on Windows:
	1.Install Node.js and NPM
	2.Open a Command Prompt or PowerShell
	3.Install Create React App: npm install -g create-react-app
	4.Create a New React App: npx create-react-app my-react-app
	5.Navigate to the App Directory: cd my-react-app
	6.Start the Development Server: npm start

	-Checking the NPM Version:
	1.npm -v
	2.# Example output 6.14.14
	3.node -v
	  npm -v
